## はじめに

Claude Codeを使う際に覚えておくと便利なキー操作をまとめました。これらのショートカットを覚えると、作業効率がグッと上がります。

## 起動・終了

| 操作 | キー・コマンド | 説明 |
|------|---------------|------|
| 確認スキップで起動 | `claude --dangerously-skip-permissions` | 起動時の確認をスキップ |
| 終了 | `Ctrl+C` | Claude Codeを終了 |
| 過去の会話を再開 | `claude --resume` | 前回の続きから再開 |

## 操作中のショートカット

| 操作 | キー | 説明 |
|------|------|------|
| 処理のキャンセル | `Esc` | 実行中の処理を中断 |
| 改行 | `Ctrl+J` | プロンプト入力中に改行を挿入 |
| モード切替 | `Shift+Tab` | 入力モードを切り替え |
| 過去の指示に戻る | `Esc`を2回 | 以前のユーザー指示を呼び出す |
| 詳細表示 | `Ctrl+R` | 折りたたまれた出力を展開 |

## 特殊な入力記号

| 記号 | 用途 | 例 |
|------|------|-----|
| `#` | メモリ（CLAUDE.md）に追記 | `# この情報を覚えておいて` |
| `!` | bashコマンドを実行 | `!ls -la`でファイル一覧表示 |
| `@` | ファイル・フォルダを指定 | `@src/index.ts`で内容を読み込む |

## まとめ

Claude Codeのキー操作を覚えると、ターミナルから離れずに素早く作業できます。特に以下の3つは使用頻度が高いので、ぜひ覚えてみてください：

- `Esc`：処理のキャンセル
- `!`：bashコマンドの実行
- `@`：ファイル指定

慣れてくると手放せなくなりますよ！
