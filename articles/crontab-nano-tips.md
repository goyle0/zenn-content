## はじめに

Linuxでcrontab（定期実行の設定ファイル）を編集するとき、デフォルトではviエディタが開くことが多いです。

viに慣れていない方にとっては、操作が難しく感じることもあるでしょう。

この記事では、**nanoエディタでcrontabを開く方法**と、**nanoで使える便利なショートカット**を紹介します。

---

## crontabをnanoで開く方法

通常、`crontab -e`を実行するとデフォルトのエディタ（多くの場合vi）が開きます。

nanoで開きたい場合は、以下のコマンドを使います。

```bash
EDITOR=nano crontab -e
```

### コマンドの意味

| 部分 | 説明 |
|------|------|
| `EDITOR=nano` | 一時的にエディタをnanoに設定する |
| `crontab -e` | crontabを編集モードで開く |

このコマンドを実行すると、nanoエディタでcrontabが開きます。

---

## nanoの便利なショートカット

nanoで編集中に使える便利なショートカットを紹介します。

### 全内容を一括で削除する

```
Alt + T
```

ファイルの内容をすべて削除したいときに使います。

### 操作を元に戻す（Undo）

```
Alt + U
```

間違えて削除してしまったときなど、一つ前の状態に戻せます。

---

## よく使うnanoのショートカット一覧

| ショートカット | 機能 |
|---------------|------|
| `Ctrl + O` | ファイルを保存する |
| `Ctrl + X` | nanoを終了する |
| `Ctrl + K` | 現在の行を切り取る |
| `Ctrl + U` | 切り取った行を貼り付ける |
| `Ctrl + W` | 文字列を検索する |
| `Alt + T` | 全内容を削除する |
| `Alt + U` | 元に戻す（Undo） |

---

## まとめ

- crontabをnanoで開くには `EDITOR=nano crontab -e` を使う
- `Alt + T` で全内容を一括削除できる
- `Alt + U` で操作を元に戻せる

viが苦手な方は、nanoを使うと編集がスムーズになります。ぜひ試してみてください。
