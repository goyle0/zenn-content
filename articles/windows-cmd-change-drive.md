## やりたかったこと

エクスプローラーのアドレスバーに `cmd` と入力してコマンドプロンプトを開く方法は便利ですが、**管理者権限では開けません。**

そこで、管理者としてコマンドプロンプトを開いた後、Dドライブのフォルダに移動したかったのですが、普通の `cd` コマンドでは移動できませんでした。

## うまくいかなかった例

管理者でコマンドプロンプトを開くと、最初は `C:\Windows\System32` にいます。

ここから `cd` でDドライブに移動しようとしても…

```cmd
C:\Windows\System32> cd D:\YMM
C:\Windows\System32>
```

**何も起きません。** カレントディレクトリがCドライブのままです。

## 原因

`cd` コマンドは、**同じドライブ内のフォルダ移動**にしか対応していません。

CドライブからDドライブのように**別のドライブに移動する場合**は、`/d` オプションが必要です。

## 解決方法

### 方法1: `/d` オプションを使う（おすすめ）

```cmd
cd /d D:\YMM
```

`/d` を付けるだけで、**ドライブの切り替えとフォルダ移動を同時に**やってくれます。

### 方法2: ドライブを先に切り替える

```cmd
D:
cd \YMM
```

先に `D:` と入力してドライブを切り替えてから、`cd` でフォルダに移動する方法です。2行必要ですが、こちらでもOKです。

## まとめ

| やりたいこと | コマンド |
|------------|---------|
| 同じドライブ内で移動 | `cd フォルダパス` |
| 別ドライブに移動 | `cd /d ドライブ:\フォルダパス` |
| ドライブだけ切り替え | `D:` (ドライブ文字 + コロン) |

**`cd /d`** を覚えておけば、どこからでも一発で移動できます！
